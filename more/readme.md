# 精华机制说明
1. 第18赛季起才有灵感折扣，6折或8折<br>
2. 1-12赛季排位珍宝内无排位头像框兑换卡<br>
3. 自2023年8月31日版本（第29赛季）更新后：常规赛季精华抽取时，将最多连续获得2次相同的奇珍品质时装 <br>
4. 精华概率是对官方公开的概率进行模拟，与官方概率<font color="red"><b>并不完全一致</b></font><br>
5. 本模拟器模拟的紫皮概率：设P(n)代表第n抽时的独立概率，P(n)= 0.1176% * n - 0.25%（n＞2），第1～2抽概率接近于0<br>
6. 本模拟器模拟的金皮概率：P(n)= 0.0063% * n - 0.032%（n＞5），第1～5抽概率接近于0<br>
7. 精华头像80/130发二选一实现机制麻烦，还是按照80、130分别获得<br>
